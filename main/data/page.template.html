О системе мониторинга

Технические характеристики

  • Микроконтроллер: ESP32
  • Датчики: DHT22, MQ-135, BMP280, PMS5003
  • Интерфейс: Веб-сервер на порту 80
  • Протокол: HTTP REST API
  • Обновление данных: Каждые 5 секунд
  • Графики: Canvas для визуализации

Мониторируемые параметры

  • Температура: -40°C до 80°C
  • Влажность: 0% до 100%
  • CO₂: 0-5000 ppm
  • CO: 0-1000 ppm
  • LPG: 0-10000 ppm
  • NH₃: 0-500 ppm
  • PM1.0 / PM2.5 / PM10: мкг/м³

Пороги предупреждений

CO₂ > 1000 ppm Предупреждение
CO > 50 ppm Опасность
LPG > 1000 ppm Взрывоопасно
PM2.5 > 35 мкг/м³ Нездоровый
PM10 > 150 мкг/м³ Опасный
